# Postagem de Blog

Aplicação de publicação de posts, feita com NodeJS e Typescript.

## Primeiros passos

### Instalação
 `npm install`

## Scripts disponíveis 

### Para rodar o projeto em dev mode
   `npm run start`


Após isso, vá até [http://localhost:80](http://localhost:80) para ver a aplicação em seu navegador.
